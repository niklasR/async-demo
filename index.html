ASYNC EXAMPLES

Just some examples of async and its capabilities

map

Code 
var names = ['niklas','anna','thomas'];

async.map(
  names,
  (name, callback) => {
    var newName = name.replace('a', 'e');
    callback(null, newName);
  },
  (error, result) => {
    console.log(result);
  }
);
Output 
["nikles", "enna", "thomes"]
